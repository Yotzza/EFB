# Documentation Index

This directory is the source of truth for Codex and human collaborators. Keep `AGENTS.md` short and use this index to find the deeper context.

## Product

- `product/game-vision.md`: core game idea, creative pillars, and feature fit test.
- `product/feature-ledger.md`: current feature inventory and the required update point for feature-facing changes.

## Design

- `design/art-direction.md`: pixel-art palette, setting, asset rules, and visual anti-goals.

## Engineering

- `ARCHITECTURE.md`: current Godot scene/script structure and dependency direction.
- `agent-harness.md`: how Codex should use this repo and how recommendations from the OpenAI harness article are applied here.
- `QUALITY_SCORE.md`: lightweight health grades and next quality moves.
- `tech-debt-tracker.md`: known debt and suggested fixes.

## Plans

- `exec-plans/active/`: checked-in plans for larger in-progress work.
- `exec-plans/completed/`: finished plans kept for design history.
- `templates/exec-plan-template.md`: starter format for complex work.
- `templates/feature-ledger-entry.md`: starter row for feature documentation updates.
