# Game Vision

Escape From Beyoslavia is a top-down 2D survival game set in a fictional post-apocalyptic Eastern Europe.

The player should feel vulnerable, resourceful, and grounded in a harsh but believable world. Systems should favor survival tension, scarcity, scavenging, careful risk-taking, and reputation consequences over arcade power fantasy.

## Creative Pillars

- Top-down 2D survival gameplay.
- Gritty post-apocalyptic Eastern European setting.
- Muted, realistic, low-saturation color palette.
- 16-bit pixel-art style with clean pixel edges and minimal anti-aliasing.
- Consistent tile-based environments.
- Soft ambient lighting, not glossy or high-fantasy lighting.
- Interface and assets should feel worn, utilitarian, and grounded.

## Gameplay Promise

The core loop should move toward:

1. Prepare from a base or safe hub.
2. Enter dangerous locations.
3. Scavenge supplies and make tradeoffs.
4. Survive threats, hunger, injury, exposure, or social conflict.
5. Return, recover, trade, improve skills, and face consequences.

Existing systems already point toward player money, reputation, skill points, level, and skills such as strength, endurance, metabolism, agility, luck, and barter.

## Feature Fit Test

A new feature fits the game when it strengthens at least one of these:

- Survival pressure.
- Scarcity or meaningful resource choice.
- Grounded Eastern European post-apocalyptic atmosphere.
- Player vulnerability and consequence.
- Readable top-down 16-bit presentation.
- Long-term character growth without breaking the survival tone.

Features that feel neon, heroic, magic-heavy, overly clean, comedic, or high-saturation should be redesigned before implementation.

## Documentation Rule

When a feature is added, changed, or removed, update `docs/product/feature-ledger.md`. If the change modifies any creative pillar, update this file in the same change.
